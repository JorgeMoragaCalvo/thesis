# Task eight

## Task. Descripción de cómo se aplicó la metodología seleccionada

Esto según la metodología que se usó. Para el caso de SCRUM se debe describir en detalle cómo se desarrollaron cada uno de los Sprints (actividades, artefactos y roles). La idea de este capitulo es que el lector le quede claro que en el desarrollo de esta memoria se usó la metodología propuesta, en este caso Scrum.


### 8.1 Sprint 1: “Nombre del Sprint”

Se debe incluir información respecto de:
- ¿Cuál era el objetivo principal?
- ¿Qué se buscaba entregar al final del Sprint?

Ejemplo:
- Desarrollar y probar el módulo de registro de usuarios y validación de credenciales para el sistema de préstamos.

Lista de historias de usuario seleccionadas del Product Backlog, con sus prioridades y Story Points.

Ejemplo:
- HU01: Como docente, quiero registrar un préstamo para poder reservar un proyector.
- HU02: Como administrador, quiero consultar los préstamos activos para gestionar la disponibilidad.

Describir brevemente las actividades que se realizaron: desarrollo, pruebas, reuniones clave, documentación, diseño de interfaz, etc.

Ejemplo:
- Se desarrolló el componente frontend para ingresar un nuevo préstamo.
- Se implementaron los endpoints REST en Spring Boot.
- Se realizaron pruebas unitarias y de integración.
- Se realizó una revisión de código al cierre del Sprint.

**¿Qué se logró al final del Sprint?** **¿Qué funcionalidad quedó lista y probada?**

Ejemplo:
- Se entregó una versión funcional del registro de préstamos, integrada y probada, accesible desde la vista del docente.

**¿Qué problemas surgieron? ¿Cómo se abordaron?**
Ejemplo:
Se presentaron dificultades para integrar el frontend con la API debido a cambios en el modelo de datos.
Se decidió postergar una historia para el siguiente Sprint.

**¿Qué se dijo en la Sprint Review y Retrospective? ¿Qué mejoras se propusieron para el equipo?**

Ejemplo:
- Se validó la funcionalidad con el Product Owner, quien aprobó el módulo con pequeñas observaciones. En la retrospectiva, el estudiante acordó mejorar la planificación.
