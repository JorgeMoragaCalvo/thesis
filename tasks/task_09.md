# Task nine

## Task. Conclusiones y trabajo futuro

### 9.1 Cumplimiento de objetivos específicos.

Transcribir textualmente cada objetivo específico. A continuación de cada objetivo específico describir cómo se cumplió dicho objetivo. Se deben dar evidencias concretas tomando información de los capítulos respectivos. Por ejemplo, si el objetivo específico hace referencia a los requisitos, entonces se debe mencionar el número de requisitos funcionales, el número de requisites no funcionales, cómo se obtuvieron y definieron esos requisitos, etc. Toda esta información se puede obtener de los capítulos del documento.
